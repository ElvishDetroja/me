Netflix

VISIT THE WEBSITE
COLLPASE
  • Data is retrieved from the The Movie Database (TMDB) API for this React-built Netflix clone.
  • Web app also includes the YouTube search API, which displays the appropriate movie trailer for each movie.
  • The platform recommends movies based on the user's previous watched movies that have a high probability of being liked by the user.
  • The primary function of this clone is Advance Search, which allows users to search using a variety of filter choices, including movie genres, release dates, runtimes, languages, movies from a certain region, and whether movies are accessible on other over-the-top (OTT) platforms.
  • Only the login feature is functional at the moment; the sign-up feature needs backend development, which I am currently working on.
